# 知南 第二批设计改造计划

## 需求汇总（用户确认）

| # | 功能 | 说明 |
|---|------|------|
| 1 | 改名 | UI 显示从「南大论坛」→「知南」，路由 /forum 不变 |
| 2 | 登录门槛 | 列表/卡片视图也截断（只显示少量），帖子正文模糊遮罩+提示登录 |
| 3 | 首页板块 | 上方：精选热帖（筛选逻辑）；下方：实时帖子（按时间流） |
| 4 | 动效升级 | 所有悬浮/切换/过渡加明显动态效果（不是微妙的） |
| 5 | 帖子详情 | 小退出键 + 相关帖子推荐（按相关性） |
| 6 | 测试数据 | 多写几个帖子进数据库，看真实效果 |
| 7 | 回到顶部 | 滚动一定距离后浮现剪影按钮，点击平滑滚动+动效 |
| 8 | 命令面板 | 替代底部 footer，左下浮动按钮 + Ctrl/⌘K 触发，导航+设置+主题切换 |
| 9 | 登录页恢复 | 之前有背景动效的登录首页不见了，要恢复 |

## 实施分段

### Part A — 基础改动（改名 + 测试数据 + 登录页恢复）
1. 全局 UI 文本「南大论坛」→「知南」
2. Prisma seed 脚本写入 15-20 条测试帖子（多主题多标签）
3. 检查登录页备份，恢复带背景动效的版本

### Part B — 登录门槛 + 首页重构
4. 未登录截断逻辑：列表/卡片只显示 3-5 条，后面模糊遮罩 + 登录提示
5. 帖子详情页：正文显示前 2 段，后面渐变模糊 + 登录解锁按钮
6. 首页分两块：精选热帖（按 favorites+replies 排序取 top）+ 实时帖子（按 createdAt）

### Part C — 交互组件
7. 回到顶部按钮（scroll > 600px 浮现，点击 smooth scroll + scale 动画）
8. 命令面板组件（⌘K 触发，导航列表+主题切换+设置项）
9. 左下浮动按钮（剪影图标，点击打开命令面板）
10. 移除现有 footer

### Part D — 帖子详情增强 + 动效
11. 帖子详情页：小退出键（左上/右上，返回列表）
12. 相关帖子推荐（按 tags 重叠度排序，底部横向卡片组）
13. 全局动效升级：hover scale/glow、视图切换 fade、列表项 stagger 入场

## 精选热帖筛选逻辑

```
score = favorites * 3 + replies * 2 + (7天内 ? 10 : 0)
取 score 最高的 6 条
```

## 相关帖子推荐逻辑

```
对当前帖子的 tags，查找其他帖子中 tag 重叠数最多的，取 top 3
若 tag 为空，取同 section 最新 3 条
```
