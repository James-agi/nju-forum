# Part B — 登录门槛 + 首页重构：技术细节

## B4. 未登录截断（列表/卡片视图）

实现方式：Server Component 中检测 session（`getServerSession`）

未登录时：
- 列表视图：只渲染前 4 条帖子，第 5 条位置放模糊遮罩层
- 卡片视图：只渲染前 3 张卡片，后面放遮罩
- 标签/主题视图：同理，限制可见数量

遮罩层组件 `<LoginGate>`：
```tsx
// src/components/auth/login-gate.tsx
"use client"
// 渐变模糊 + 居中文案 + 登录按钮
// backdrop-filter: blur(8px)
// 渐变：从透明到 background 色
// 文案：「登录后查看全部内容」
// 按钮：跳转 /login
```

## B5. 帖子详情截断

未登录时 post detail 页面：
- 正文只显示前 ~300 字（约 2 段）
- 后面接渐变模糊遮罩
- 遮罩上方显示：「▪ 后续为登录内容 · 完整收录：{帖子标题}」
- 按钮：「登录解锁 →」

已登录：正常显示全文。

## B6. 首页分块

将 /forum 默认视图（list）改为双板块布局：

```
┌─────────────────────────────────────┐
│  精选热帖（横向卡片，3-4 条）        │
│  score = favorites*3 + replies*2     │
│         + (7天内 ? 10 : 0)           │
├─────────────────────────────────────┤
│  实时动态（纵向列表，按时间倒序）     │
│  PostListRow 复用，无限滚动或分页     │
└─────────────────────────────────────┘
```

精选热帖：
- 独立查询，不受视图切换影响
- 横向滚动或 grid 展示，带序号和热度标签
- 卡片样式区别于普通卡片（加 🔥 或边框高亮）

实时动态：
- 就是现有的 list 视图按 createdAt desc
- 标题栏显示「实时」+ 呼吸灯小圆点动画
