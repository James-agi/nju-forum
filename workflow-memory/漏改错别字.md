# 工作流记忆：漏改错别字

## 日期
2025-08-11

## 事件
用户指出"南那助手"应改为"南哪助手"，我第一次修改时漏改了。

## 错误过程
1. 用户指出"不是'那'是'哪'"
2. 我修改了正文中的"南那助手"→"南哪助手"
3. 但漏改了标题、summary、sourceExcerpt等处
4. 用户再次指出"没改呀"

## 错误原因
- **没有全面检查**：只改了body中的内容，漏改了其他地方
- **没有区分修改范围**：sourceExcerpt是逐字原文，不应该改；但标题、summary需要改
- **修改后没有重新检查**：改完就提交了，没有通读一遍

## 正确做法
1. **修改时列出所有需要修改的地方**：标题、summary、body、sourceExcerpt等
2. **区分哪些需要改，哪些不需要**：
   - 需要改：标题、summary、body（我的创作内容）
   - 不需要改：sourceExcerpt（逐字原文，保持原样）
3. **修改后重新检查一遍**：通读所有内容，确认没有遗漏

## 检验方法
- 改完后用搜索功能查找旧词，确认全部替换
- 区分"创作内容"和"引用内容"，引用内容保持原样
