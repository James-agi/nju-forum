# 工作流记忆：爬虫误判页面类型

## 日期
2025-08-11

## 事件
用户提供语雀URL https://www.yuque.com/greatnju/q-a2.0/gw3phd，要求爬取内容。

## 错误过程
1. 用webfetch抓取，只返回标题
2. 用Playwright抓取，获得完整内容
3. **错误**：看到页面侧边栏有目录结构，误判为"目录页"
4. 实际上该URL是具体文档《南哪助手长期接受同学们投稿》

## 错误原因
- **混淆了文档内容和页面UI元素**：语雀文档页面会显示侧边栏目录，但这不代表它是目录页
- **没有区分"页面结构"和"文章内容"**：侧边栏是UI元素，不是文章主体
- **过早下结论**：看到目录结构就断定是目录页，没有仔细确认爬取到的实际内容

## 正确做法
1. **先确认爬取到的实际文章内容**：看页面标题和正文
2. **区分UI元素和内容**：侧边栏、导航栏是UI，不是文章主体
3. **不要被页面结构迷惑**：具体文档也可能有侧边栏目录

## 检验方法
- 页面标题是文档标题（如"南哪助手长期接受同学们投稿"）
- 正文内容是文章主体
- 侧边栏是UI导航，不是文章内容
